# Este directorio contiene los archivos generados por el scraper de stock

Los archivos aquí son generados automáticamente y no deben ser versionados en Git.

## Tipos de archivos:

- `stock-machines-*.json` - Datos completos de todas las máquinas
- `stock-summary-*.json` - Resumen agregado por producto
- `stock-machines-*.csv` - CSV con todos los datos
- `stock-summary-*.csv` - CSV con el resumen
- `lista-carga-*.txt` - Lista de carga imprimible para la furgoneta
